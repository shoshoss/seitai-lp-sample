# AGENTS.md

## プロジェクト概要

このプロジェクトは、整体・鍼灸院向けの 1 ページ LP 制作サービスのサンプルサイトです。

目的は、技術には自信があるものの、初めての人に院の良さが伝わらず、新規予約が安定していない小規模自費整体・鍼灸院に向けて、予約前の不安を減らす LP 制作サービスを紹介し、無料相談につなげることです。

## ターゲット

- 小規模自費整体・鍼灸院
- 開業 3〜7 年目
- 院長 1 人〜少人数運営
- 技術には自信がある
- 紹介やリピートはあるが、新規予約が安定していない
- ホームページ、SNS、Google マップ、LINE 予約などの導線が整理できていない

## LP のゴール

最終ゴールは「無料相談」への誘導です。

ユーザーにいきなり契約を迫るのではなく、現在のホームページ・SNS・予約導線について気軽に相談できる印象を作ってください。

## LP の心理設計

以下の流れを意識してください。

1. 認知：ファーストビューで一瞬で刺す
2. 興味：ベネフィットで感情を動かす
3. 比較：料金、制作範囲、FAQ、他との違いで納得してもらう
4. 相談：CTA で最後の一押しをする
5. 継続：公開後のサポートや次のステップを明確にする

## LP 構成

以下の順番で構成してください。

1. ヘッダー
2. ファーストビュー
3. 悩みチェックリスト

5. 初めての患者さんが予約前に知りたいこと

## コーディング方針

- HTML/CSS のみで作成する
- JavaScript は使わない
- GitHub Pages で公開しやすい構成にする
- index.html はリポジトリ直下に置く
- CSS は css/style.css にまとめる
- レスポンシブ対応を必ず行う
- スマホ表示を優先する
- セマンティック HTML を意識する
- class 名は分かりやすくする

## 禁止表現

整体・鍼灸院向けの LP であるため、以下のような断定表現や過度な表現は使わないでください。

- 必ず予約が増える
- 絶対に集客できる
- 売上が必ず上がる
- 症状が治る
- 必ず改善する
- 効果を保証する
- 地域 No.1
- 医学的な効果を断定する表現

## 推奨表現

以下のような表現を使ってください。

- 予約前の不安を減らす
- 院の良さを分かりやすく伝える
- 施術内容・料金・予約方法を整理する
- 初めての方が安心して相談しやすい導線を作る
- SNS や Google マップから予約までの流れを整える
- 公開して終わりではなく、育てる LP

## CTA 方針

CTA は強く売り込むのではなく、相談しやすさを重視してください。

例：

- 無料で予約導線を相談する
- まずは無料相談してみる
- ホームページ・SNS 導線を相談する

CTA 付近には以下を明記してください。

- 無料相談 OK
- 無理な営業なし
- 文章がまとまっていなくても OK
- ヒアリングしながら一緒に整理
- 現在のホームページや SNS がなくても相談 OK

## 作業時の注意

- 変更前に目的を確認する
- 大きな変更は一度に行わず、セクション単位で進める
- 変更後は、どのファイルをどう変更したか説明する
- スマホ表示で崩れないか確認する
- CTA リンクは #contact に統一する
- ダミーリンクや不要な外部リンクは入れない
