# 整体・鍼灸院向け LP サンプル

整体・鍼灸院向けに、初めての方が安心して予約できるよう、施術内容・料金・予約方法・院の強みを 1 ページに整理する LP です。

## 概要

技術には自信があるものの、初めての方に院の良さが伝わりにくい小規模な整体・鍼灸院を想定して制作しました。

強く売り込むのではなく、予約前の不安を減らし、現在のホームページ・SNS・Google マップ・LINE 予約などの導線を整理することを目的にしています。

## 制作目的

- 初めての患者さんが予約前に知りたい情報を分かりやすくまとめる
- 院長の考え方や施術方針が伝わる構成にする
- 料金、予約方法、よくある質問を整理し、相談前の迷いを減らす
- 無料相談へ自然につながる導線を作る

## ターゲット

- 小規模な自費整体・鍼灸院
- 開業 3〜7 年目の院長
- 技術には自信があるが、新規予約が安定していない
- ホームページ、SNS、Google マップ、LINE 予約の導線を整理したい方

## LP のゴール

最終ゴールは「無料相談」への誘導です。

いきなり契約を迫るのではなく、現在の予約導線や情報発信について気軽に相談できる印象を重視しています。

## 使用技術

- HTML
- CSS
- JavaScript なし
- GitHub Pages で公開しやすい静的構成

## 主なセクション

- ファーストビュー
- 悩みチェックリスト
- 予約につながらない原因
- 初めての患者さんが予約前に知りたいこと
- LP で整えること
- ベネフィット
- 制作内容
- 他との違い
- 料金プラン
- よくある質問
- 無料相談エリア
- 公開後サポート
- プロフィール

## 工夫した点

- 院長目線だけでなく、初めての患者さん目線も入れた構成
- 「施術内容」「料金」「予約方法」「院の強み」を 1 ページで確認できる情報設計
- 料金プランを 1 つに絞り、選択の迷いを減らす設計
- 無料相談前に確認できること、準備するもの、相談後の流れを明記
- スマホで読み進めやすい余白、カード、1 列レイアウトの調整
- 強い煽りや断定表現を避けた、誠実な文章設計

## デザイン方針

清潔感、安心感、信頼感、やさしさを意識しています。

白とオフホワイトをベースに、淡い青を信頼感、ミント系を安心材料の表現に使用しました。派手な装飾よりも、読みやすさと相談しやすさを優先しています。
