# 整体・鍼灸院向けLP制作サービス サンプル

小規模な自費整体・鍼灸院向けに、予約前の不安を減らす1ページLP制作サービスのサンプルサイトです。

## 目的

技術には自信があるものの、初めての方に院の良さや予約方法が伝わりにくい整体・鍼灸院に向けて、施術内容・料金・予約導線を整理するLP制作サービスを紹介し、無料相談につなげます。

## ファイル構成

```text
.
├── index.html
├── css/
│   └── style.css
├── docs/
│   ├── requirements.md
│   └── lp-structure.md
└── README.md
```

## 技術方針

- HTML/CSSのみ
- JavaScriptなし
- GitHub Pagesで公開しやすい構成
- `index.html` はリポジトリ直下
- CSSは `css/style.css` に集約
- スマホ表示を優先したレスポンシブ対応
- CTAリンクは `#contact` に統一

## 公開方法

GitHub Pagesで公開する場合は、リポジトリの Pages 設定から公開ブランチを選択してください。トップページとして `index.html` が表示されます。
