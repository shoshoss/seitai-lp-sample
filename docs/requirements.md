# 要件定義

## プロジェクト概要

整体・鍼灸院向けの1ページLP制作サービスを紹介するサンプルサイトです。

目的は、小規模な自費整体・鍼灸院に向けて、院の良さや予約導線を分かりやすく整理するLP制作サービスを伝え、無料相談へつなげることです。

## ターゲット

- 小規模自費整体・鍼灸院
- 開業3〜7年目
- 院長1人〜少人数運営
- 技術には自信がある
- 紹介やリピートはあるが、新規予約が安定していない
- ホームページ、SNS、Google マップ、LINE予約などの導線が整理できていない

## LPのゴール

最終ゴールは無料相談への誘導です。

いきなり契約を迫るのではなく、現在のホームページ・SNS・予約導線について気軽に相談できる印象を作ります。

## 必須要件

- HTML/CSSのみで作成する
- JavaScriptは使用しない
- GitHub Pagesで公開しやすい構成にする
- `index.html` をリポジトリ直下に置く
- CSSは `css/style.css` にまとめる
- スマホ対応を行う
- 画像がなくても成立するデザインにする
- CTAリンクは `#contact` に統一する
- ファーストビュー、悩み、原因、ベネフィット、制作内容、料金、FAQ、CTAを含める

## 表現方針

強い煽りや断定表現は避け、以下のような表現を中心にします。

- 予約前の不安を減らす
- 院の良さを分かりやすく伝える
- 施術内容・料金・予約方法を整理する
- 初めての方が安心して相談しやすい導線を作る
- SNSやGoogle マップから予約までの流れを整える
- 公開して終わりではなく、育てるLP

## 禁止表現

- 必ず予約が増える
- 絶対に集客できる
- 売上が必ず上がる
- 症状が治る
- 必ず改善する
- 効果を保証する
- 地域 No.1
- 医学的な効果を断定する表現

## デザイン方針

- 清潔感
- 信頼感
- やさしさ
- 誠実さ
- スマホで読みやすい
- 余白をしっかり取る
- 白、淡い青、薄いグレーを中心にする
- 派手すぎる装飾を避ける
