# LP構成

## 心理設計

1. 認知：ファーストビューで対象者と価値をすぐ伝える
2. 興味：悩みとベネフィットで自分ごと化する
3. 比較：制作内容、違い、料金、FAQで納得材料を出す
4. 相談：無料相談CTAで相談しやすい空気を作る
5. 継続：公開後サポートで育てるLPであることを伝える

## セクション一覧

1. ヘッダー
   - サービス名
   - 制作内容、料金、FAQ、無料相談へのナビゲーション

2. ファーストビュー
   - 整体・鍼灸院専門のLP制作であること
   - 予約前の不安を減らし、導線を整えること
   - CTAリンクは `#contact`

3. 悩みチェックリスト
   - 新規予約が安定しない
   - ホームページやSNS導線が整理できていない
   - 施術内容や料金の伝え方に迷っている

4. 予約につながらない原因
   - 初めての方は予約前の不安を確認している
   - 情報が点在すると予約前に迷いが残りやすい

5. 初めての患者さんが予約前に知りたいこと
   - 相談できる悩み
   - 施術の流れ
   - 料金と予約方法

6. LPで整えること
   - 伝える内容の整理
   - スマホで読みやすい設計
   - 予約までの導線作り

7. ベネフィット
   - 院の良さが伝わりやすくなる
   - 問い合わせ前の不安を減らす
   - 導線を見直すきっかけになる

8. 制作内容
   - 構成作成
   - 文章整理
   - HTML/CSS制作
   - スマホ対応
   - 公開前チェック

9. 他との違い
   - 小規模院向け
   - 医療広告に配慮
   - 公開後も育てる前提

10. 料金プラン
    - LP制作ベーシック
    - 価格は目安として表示
    - 無料相談後に提案

11. 制作の流れ
    - 無料相談
    - ヒアリング
    - 構成・制作
    - 確認・公開

12. よくある質問
    - ホームページやSNSがなくても相談できるか
    - 文章がまとまっていなくてもよいか
    - 画像が少なくても制作できるか
    - 公開後の修正相談ができるか

13. 無料相談CTA
    - 無料相談OK
    - 無理な営業なし
    - ヒアリングしながら一緒に整理
    - CTAリンクは `#contact`

14. 公開後サポート
    - 公開後に内容や導線を見直せること
    - 育てるLPとして運用すること

15. プロフィール
    - 小規模院の伝わり方を整える制作者として紹介

16. フッター
    - サービス名
    - 無料相談へのリンク
